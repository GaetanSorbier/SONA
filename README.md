# SONA 
## SONA (Save Obsidian Notes Automation)

## Run Locally

Clone the project

```bash
  git clone https://obsidian-notes
```

Go to the project directory

```bash
  cd obsidian-notes
```

Give execution permissions to the script 

```bash
    chmod +x sync_notes.sh
```
